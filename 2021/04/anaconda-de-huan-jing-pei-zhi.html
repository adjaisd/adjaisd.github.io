Anaconda的环境配置

安装Anaconda

  • 首先，下载Anaconda的安装包

    •  第一种方法，直接去官网下载，点击这里，进入官网，速度可能稍慢；
    •  第二种方法，建议在镜像里下载，这样稍快一些，点击这里，进入镜像，选择对应的版本下载。

  • 接着，安装Anaconda，这里以Anaconda-Windows-x64版本介绍

    1.点击next；


    2.点击I Agree


    3.选择All users，点击next


    4.安装路径可以更改一个，便于日后清理残留
    如果C盘空间不够，建议在另一个盘里创建一个目录，安装在创建的目录里，点击next
     这里，采用默认


    5.接下来，是否添加路径环境变量，可勾选也可不勾选；
     勾选的话，就不用了自己配置
     不勾选的话，需要手动添加路径;
    这里，不勾选第一项，点击install


    6.等待安装


    7.安装完后，点击next


    8.点击finish

    ![8](Anaconda的环境配置/Anaconda的环境配置@8.webp)

 至此，安装Anaconda就完成了

配置环境

  1. 配置步骤
     比如，我的三个路径分别是
D:\Anaconda
D:\Anaconda\Scripts
D:\Anaconda\Library\bin
D:\Anaconda\Library\usr\bin
D:\Anaconda\Library\mingw-w64\bin

 其中，我的Anaconda的安装路径为 D:\Anaconda

  安装路径，安装路径下的Scripts与Library\bin路径都是必要的。

 上述三个环境变量都是通过：此电脑—右键—高级系统设置—环境变量—系统变量—双击path—新建这两个变量即可。

  1. 检查是否配置成功

  • 检测python
     进入cmd，输入python,没报错就可以


  • 检测conda
     进入cmd，输入conda，出现对conda命令的提示即表示成功。


配置Anaconda源

  • 由于通过Anaconda自带的源，可能下载第三方库非常慢，因此，需要配置下国内的源；

  • 以我的配置为例

     我配置的源有以下几个，当然大家也可以自己找其他的源
     cmd下，键入conda config –show channels，我的显示如下。

channels:
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
- http://mirrors.ustc.edu.cn/anaconda/cloud/conda-forge/
- http://mirrors.ustc.edu.cn/anaconda/pkgs/free/
- http://mirrors.ustc.edu.cn/anaconda/pkgs/main/
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
- http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2
  • 配置源
    •  通过conda config –add channels [url] 命令，就可以添加源了。
    •  比如，如果想把http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/ 添加到自己的源里，
       进入cmd下，输入 conda config –add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
conda config --add channels http://mirrors.ustc.edu.cn/anaconda/cloud/conda-forge/
conda config --add channels http://mirrors.ustc.edu.cn/anaconda/pkgs/free/
conda config --add channels http://mirrors.ustc.edu.cn/anaconda/pkgs/main/
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
conda config --add channels http://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/msys2

conda config --set show_channel_urls yes

  • 检察源
     cmd下通过conda config –show channels检测已经配置的源。


  • 删除源
     通过conda config –remove channels [url]来删除源。

补充Linux下配置Miniconda

 有时nodejs旧版本的包会依赖python2，但又需要用到python3，采用Miniconda是个不错的选择，可参考如下脚本:

rm /tmp/Miniconda3*.sh -rf
wget https://repo.continuum.io/miniconda/Miniconda3-latest-Linux-x86_64.sh -P /tmp
ls -lht /tmp/ | grep "Miniconda"
bash /tmp/Miniconda3-latest-Linux-x86_64.sh
# 略去安装步骤
cat << EOF  > /etc/profile.d/conda.sh
export CONDA_HOME="/usr/local/miniconda3"
export PATH="\${CONDA_HOME}/bin:\${CONDA_HOME}/condabin:\$PATH"
EOF

chmod +x /etc/profile.d/conda.sh
source /etc/profile.d/conda.sh
conda -V
