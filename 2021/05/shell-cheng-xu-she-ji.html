Shell程序设计

 回顾大二时学操作系统时写的这篇文章，发现错漏很多，去掉无用的话;
 书找不到了，就参照菜鸟学吧

基本语法

变量

 略，命名和其他编程语言规则类似;
 可参考这篇文章:https://www.cnblogs.com/liang-io/p/9825363.html

name="adj"
echo $name  # 等价echo ${name}和echo "$name"
readonly name # 仅在本次会话和子会话有效
# name="adja" # 报错
unset name  # 删除变量，但无法删除只读变量

字符串

name="adjaisd"
echo '$name='"${name}"  # $name=adjaisd
echo "\$name=$name" # $name=adjaisd
echo 'len='${#name}  # len=3
echo "${name:2:4}"  # jais，获取子字符串
  • 变量赋值时，如果字符串里包含空格，就必须用引号把它们括起来(不能为name = adj，而应该为name=” adj”);
  • 双引号中$运算符会生效，转移字符除外，而单引号不会;

数组

 shell仅支持一维数组;

name="adja"
names=(1 "adj" "$name" haha)
echo $names, ${names[0]} # 1, 1
echo ${#names}, ${#names[0]}, ${#names[@]}  # 1, 1, 4
echo ${#names[1]}, ${#names[2]}, ${#names[3]} # 3, 4, 4

 此外，还有通过declare -A声明的关联数组，和字典很像，下标不仅仅是数字;

declare -A site=(["google"]="www.google.com")
site["taobao"]="www.taobao.com" # 或者declare -A site=(["google"]="www.google.com" ["taobao"]="www.taobao.com")
echo site["google"] # www.google.com

注释

 常在行首用’#’.

参数传递

root@VM-16-16-ubuntu:/home/ubuntu/workspace/practice# cat test.sh 
#!/bin/bash
echo "params: "'$#,$0,$1,$2,$3 =' "$#, $0, $1, $2, $3"
echo '$* =' "$*"
for param in "$@"; do
        echo $param;
done
for param in "$*"; do
        echo $param;
done
echo "ending"
root@VM-16-16-ubuntu:/home/ubuntu/workspace/practice# ./test.sh 1 2 3 
params: $#,$0,$1,$2,$3 = 3, ./test.sh, 1, 2, 3
$* = 1 2 3
1
2
3
1 2 3
ending

 从以上实例中，$#获取参数个数，$0获取shell脚本名称，$n(n>0)获取第n个参数;
 $*与$@都可以获取所有参数，但$*将参数当作一个整体，而$@则一个一个传参数.

运算符

  1. 算数运算符

 包含’+, -, *, /, =, ==, !=’,其中 * 需要用转义字符 * 表示;

echo `expr 3 + 4` # 7
echo `expr 3 \* 4` # 12
  1. 关系运算符

 包含’-lt, -gt, -le, -ge, -eq, -ne’, 只支持数字,除非字符串的值是数字;

if [ 1 -lt 5 ]; then echo ok; fi  # ok
  1. 布尔运算符

 包含非(!),或(-o),与(-a),示例如下：

if [ ! 1 -lt 5 ]; then echo 1; else echo 2; fi # 2
if [ 1 -gt 5 -o 1 ]; then echo 1; else echo 2; fi # 1
if [ 1 -gt 5 -a 1 ]; then echo 1; else echo 2; fi # 2
  1. 逻辑运算符

 包括与(&&),或(||),其中需要两对[]将表达式括起来;

if [[ 1 -gt 5 && 1 ]]; then echo 1; else echo 2; fi # 2
if [[ 1 -gt 5 || 1 ]]; then echo 1; else echo 2; fi # 1
  1. 字符串运算符

 包含等于(=),不等于(!=),长度是否为0(-z),长度是否不为0(-n),是否为空($);

if [ "abc" = "bcd" ]; then echo 1; else echo 2; fi # 2
if [ "abc" != "bcd" ]; then echo 1; else echo 2; fi  # 1
if [ -z "abc" ]; then echo 1; else echo 2; fi # 2
if [ -n "abc" ]; then echo 1; else echo 2; fi # 1
if [ $name ]; then echo 1; else echo 2; fi # 1, 其中name=adj
  1. 文件测试运算符

 参数包括是否块设备文件(-b),字符设备文件(-c),目录(-d),普通文件(-f),可读/写/执行(-r/w/x),大小大于0(-s),存在(-e);

root@VM-16-16-ubuntu:/home/ubuntu/workspace/practice# ls -l
total 16
-rw-r--r-- 1 root root    0 Nov 22 17:25 '='
-rwxr-xr-x 1 root root   50 Nov 22 16:35  comment.sh
drwxr-xr-x 2 root root 4096 Nov 20 16:42  test
-rwxr-xr-x 1 root root  181 Nov 22 15:43  test.sh
-rw-r--r-- 1 root root  117 Nov 20 19:46  test.txt
if [ -e test.txt ]; then echo "存在"; fi # 存在
if [ -d test ]; then echo "是目录"; fi  # 是目录
if [ -r comment.sh ]; then echo "可读"; fi  # 可读
  1. test指令

 用于检查条件是否成立,以上用于if语句的都可以用test目录表示;

if test 3 -gt 2; then echo 3大于2; fi  # 3大于2
if test -e test.txt; then echo exist; fi # exist

 补充，$[a+b]可以获取数字a和b之和，$[expr]可以用于求表达式的值

printf指令

 同c语言，示例如下;

printf "name=%-5s sex=%s age=%d\n" adj male 18  # name=adj   sex=male age=18

流程控制

  1. 条件语句

 上文已出现许多，就不举例了，给出格式:

# 单分支
if condition; then
  command
  ...
fi
# 多分枝
if condition1; then
  command1
elif condition2; then
  command2
  ...
else
  commandN
fi

 如果想写成单行，需要在每一行后面添加封号并删去换行符;

if condition1; then command1; elif condition2; then command2; ...; else commandN; fi
  1. case分支语句

 感觉用的不多，用到再细查吧;

case variable in
  pattern [ | pattern] ...) command1;
  pattern [ | pattern] ...) command2;
  ...
esac
  1. 循环语句
# for循环
for variable in values; do
  command
done
# while循环
while condition; do
  command
done
# until循环
until condition; do 
  command
done

 break和continue同C.

函数

 格式如下:

[ function ] function_name(){
  command;
}

注意 1. 可以带function fun() 定义，也可以直接fun() 定义,不带任何参数。
2. 参数返回，可以显示加：return 返回，如果不加，将以最后一条命令运行结果，作为返回值。 return后跟数值n(0-255)

输入输出重定向

command > file  # 将输出重定向到 file
command < file	# 将输入重定向到 file,即file作为输入
command >> file # 将输出以追加的方式重定向到 file
n > file  # 将文件描述符为 n 的文件重定向到 file
n >> file # 将文件描述符为 n 的文件以追加的方式重定向到 file
n >& m  # 将输出文件 m 和 n 合并
n <& m  # 将输入文件 m 和 n 合并
<< tag  # 将开始标记 tag 和结束标记 tag 之间的内容作为输入

 补充:

  1. 标准输入文件(stdin)：stdin的文件描述符为0，Unix程序默认从stdin读取数据。
  2. 标准输出文件(stdout)：stdout 的文件描述符为1，Unix程序默认向stdout输出数据。
  3. 标准错误文件(stderr)：stderr的文件描述符为2，Unix程序会向stderr流中写入错误信息。
command 2>file  # 将错误写入file
command 2>>file # 错误附加到file
command > file 2>&1 # 将错误和输出合并保存到file，command >> file 2>&1
command < file1 > file2 # file1作为command输入，结果保存到file2
cat << EOF  # 用于可以对文件操作的指令，比如wc,cat，而echo不生效
  作为输入传入cat
EOF
command > /dev/null 2>&1  # 屏蔽输出与错误输出

文件包含

 类似import，被包含的文件无需可执行权限,格式如下;

. filename  # 导入filename，或者
source filename

 source需要使用bash生效，使用/bin/sh报错.

 通过|符号，左边命令的输出是下一个命令的输入

连接符

# ; 表示一个命令结束，无论对错，都会一个一个命令执行下去
# | 管道，前面的输出作为后者的输入
#|| 表示当命令执行失败后，才执行后面一个命令
#&& 表示当命令执行成功后，才能执行下一个命令
