初次刷机

 抓包困扰了我很久，时常遇到些棘手的问题，有些问题即便解决了但还会遇到新问题；

 为了更好地抓包，在入手了一部新机后，立马对旧机进行刷机，先说明一下手机是红米k20pro尊享班，机型不同的话仅作参考，下面记录下中间遇到的问题。

解锁BootLoader

  1. 开启开发者模式：设置–>我的设备–>全部参数–>MIUI版本连续点击多次；

  2. 绑定帐号和设备：设置–>更多设置–>开发者选项–>查看解锁状态–>按照提示绑定帐号和设备

  3. 解锁

 电脑下好解锁工具：https://unlock.update.miui.com，打开工具登陆帐号；

 手机关机，关机后同时长按音量下键和开机键进入fastboot模式，然后用数据线连接电脑和手机；

 这里可能连接了手机，但软件中还是”当前未连接手机”，尝试下安装USB驱动：点击软件右上角的设置–>驱动检测，提示“请重新连接手机”–>断开数据线再重接上，尝试几次；

 当检测到连接上手机后，点击解锁(注意，解锁会删除所有数据，记得备份，不要简单的备份数据，还是测试下新机中能否正常使用);

 也有可能还不能解锁，听网上说解锁前需等待几天，由于购新机等待的时间已经大于了验证时间，这个情况还没遇到过。

安装twrp

 twrp介绍直接去百度或维基看，这里说明具体的操作步骤，也可以参考论坛教参：http://www.romleyuan.com；

  1. 修改手机设置

 首先，更改下手机的部分设置，开发者选项中启动 **USB调试，USB安装，USB调试(安全设置)**，这些都要开启，后续可能都用得到；

  1. 安装手机驱动

 搜Xiaomi Flash，下载一个工具来为小米手机安装驱动，也可以去这里：https://www.xiaomiflash.com/download/ ；

 下载完成后，连接手机和电脑，打开软件，点击Driver–>安装；

  1. 使用一键安装twrp工具安装twrp

 我是在一个微信公众号找到的对应我的twrp安装工具(工具还需要自己找，对应红米k20pro尊享版twrp软件会在文末给出)；

 手机关机，然后同时长按音量下键和开机键进入fastboot界面，找到工具中的 recovery-twrp一键刷入工具.bat，关闭杀毒软件后双击”recovery-twrp一键刷入工具.bat”，接下来按照提示进行；

刷rom

 rom自带root权限，但有些软件会检测到root，再自带magisk进行伪装；

 之前买的rom是在这个网站9块多买的：http://www.romleyuan.com，安卓10自带magisk8，rom会在后面给出，也可以去这个网站看看：https://mi.fiime.cn/；

 我是参考的这个教程：http://www.romleyuan.com/lec/read?id=111；

 说一下中间注意的地方，twrp版本可能不一样，但主要的几个操作是差不多的，但有些差异；

  • 清除Data
 写在前面，有linux知识的话，就明白不是所有的目录或文件都可以删除，有些文件被删了(设备也是以文件的方式展示)，那么手机就真变砖了；

 twrp主界面–>点击”清除””–>点击”格式化Data分区”–>输入”Yes”–>回车；

 接着返回twrp主界面，先不要重启；

  • 拷贝rom到手机data分区

 连接手机和电脑，我的电脑是可以识别到手机的，进入手机文件中，拷贝rom到手机中;

 如果识别不了的话，重新刷twrp，回到上一步清除Data中；

  • 安装ROM

 回到twrp主界面–>点击安装–>选择待安装的rom包–>滑动开始安装–>重启:

注意：

 重启可能停在一个界面比较长(界面全黑，只有中间有个MIUI)，不要慌(等待5~15分钟)，如果还是不行的话就进入twrp换个适配的rom刷，实在不行的话远程刷机吧；刷机有风险，小心手机变砖。

刷入其他工具

刷magisk

 买的rom自带的magisk版本太低，想装个最新版的magisk25.2，这里就现学现用，可以参考下magisk官方安装文档：https://topjohnwu.github.io/Magisk/install.html

 这里采用自定义安装，说明如下：

1. Download the Magisk APK
2. Rename the .apk file extension to .zip, for example: Magisk-v24.0.apk → Magisk-v24.0.zip. If you have trouble renaming the file extension (like on Windows), use a file manager on Android or the one included in TWRP to rename the file.
3. Flash the zip just like any other ordinary flashable zip.
4. Reboot and check whether the Magisk app is installed. If it isn’t installed automatically, manually install the APK.
  1. 首先去下载一个magisk的apk文件；
  2. 接着将改变文件类型从.apk到.zip，
  3. 然后手机进入twrp界面，通过数据线将Magisk-25.2.zip文件拷贝到手机中；
  4. 安装过程同上安装rom：twrp主界面–>点击安装–>选择Magisk-25.2.zip文件–>安装；
  5. 重启。

 效果图如下：

 magisk25.2会把自身也给伪装起来，我这里它伪装成了Settings程序，实际上点击去还是magisk。

 在网站https://mi.fiime.cn/中，还有另一种方法刷magisk，和magisk工具一样，就是修改boot.img文件，优点操作简单，缺点只刷特定的版本magisk。

刷LSPosed

 由于手动刷太麻烦了，这里推酷安中的一个软件，名字叫“爱玩机工具箱”，具体操作就不演示了，自己琢磨下就会了；

 最后展示下效果图：

 LSPosed是通过这个软件装上的，所以还是感谢软件作者；

 先前通过官方仓库安装的LSPoded，但显示未激活，似乎还要刷补丁，还好有其他方法。

其他注意事项

 该教程只适用于红米k20pro尊享版，据说小米似乎还要注意是否支持A/B分区；

 换手机后，由于我忘了两张手机卡的pin码，只能用puk码解锁，这期间不要随便刷旧手机(绑定了银行，营业厅账户等)，可以通过旧手机登录的营业厅帐号通过服务密码找回puk码来解锁，不然还要跑营业厅一趟去要puk码；

 所以我说备份不要仅仅备份下数据，最好验证下新手机中各种应用能不能正常使用，好歹有个心理准备，不然遇到最差的情况就不好了；

 尽管中间遇到了种种问题，但还好解决了，情况不算太糟糕，花了大概半天多的时间；
