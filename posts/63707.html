手术资源的配置

【问题提出】

 某大医院向社会提供不同的医疗服务，为获得更好的社会效益和经济效益，医院必须优化其资源配置，如果资源配置不是最好的，则可能存在有的资源使用效率低，而有的资源使用过度的情况。以下面提供的外科手术数据为例，试建立一个能够帮助医院改善其资源配置，提高效益的数学模型。
 手术类型分为三类，简称为大手术（例如心脏搭桥手术）、中手术（例如胃切除手术）和小手术（例如阑尾炎手术），每种手术需的人数和费用见下表。

|手术 |主刀医师 |麻醉师 |配合医师 |器械护士 |巡回护士 |所需时间(d) |平均费用(万元) |
|:-: |:-: |:-: |:-: |:-: |:-: |:-: |:-: |:-: |:-: |
|大 |3 |1 |1 |2 |2 |1 |3 |
|中 |2 |1 |1 |1 |2 |0.5 |1.6 |
|小 |1 |1 |0 |1 |1 |0.2 |0.3 |

 当前医院的人员基本情况为：高级医师21人，普通医师44人，只有高级医师才能充当大、中手术的主刀医师；护士100人，其中只有60人可以充当器械护士；麻醉师30人。如果各种外科手术的病人足够多，问：
(1) 如何安排每天的日常手术使得经济效益为最高？
(2)若做手术的病人分布不均，如大手术并不常见，而小手术则可能比较多，要求做小手术的病人在手术完成前一直占据着医院的病床。如若床位有限，小手术要求一周内完成，否则病人要求转院，如何制定医院的每天手术安排计划？

讨论内容

 花了差不多一个多小时讨论，有了差不多的思路；
 从讨论中，发现了我写的第一题中的标准型中的错误，未考虑手术时间的影响；
 首先明确题目，第一题中强调每一天，只要考虑单独的一天，第二题中由于考虑到床位的限制(做手术的床位不能被抢占)和相互影响的每天，先假设床位非常充足，这样其实第一问和第二问就差不多了，思路如下：
 取大中小手术时间的最大公约数 0.1 天为单位，将总时间 T 天划分为 w 个0.1天，一共有 h=21 + 44 + 100 + 30=195个人，由此得到一个 h x w 的任务指派的0-1矩阵，这时只知道了某个人在某个时间段是否被分配了任务，还需要知道他做了哪一个手术中的哪一个任务，就用矩阵 K 表示吧；
 怎么表示矩阵 K，需要设两个新矩阵分别表示 某个人如果指派了任务，就要知道它被指派做了哪一个手术(是大、中还是小手术)，还有他做的手术中的哪一个任务(主刀医师,麻醉师,配合医师,器械护士,巡回护士)。这里分别用 B 和 C 表示；其中，B 的形状为 w x 3 (3代表大中小手术) ，C 的形状为 3 x 5 (5为手术的任务种类)，所以可以获得矩阵的关系为：K = B x C ( K 的形状为 w x 5)；
 到这里，就可以知道某个人在某个时间段做了某个手术中的哪个任务，用矩阵 T = A x K = A x B x C (T 的形状为 h x 5 = 195 x 5) 表示，而我们的目标是，要知道分别做了多少大中小手术，用矩阵 M (形状为 1 x 3 )表示，再根据价值矩阵 N (为常量，形状为 3 x 1)求出最终价值，优化目标函数为 max z = M x N；
 矩阵 A x B 表示某个人在某个时间段做了哪场手术，通过对时间进行遍历，统计出连续的时间段满足条件可以做的大中小手术数量，就可以得到矩阵 M 了
 接着就是各种约束了，暂未完成，后续补上…

 针对第二问，需要加上床位的限制，同样是指派问题，添加对床位的指派，假设有 n 个床位，在第一问的基础上，矩阵与变量含义同上，添加一个床位指派矩阵 D (形状为 n x w )；
 由于一个病人在一个星期得不到治疗就会离开，需要考虑时间问题，在床位不足的情况下，需要优先考虑，仍需要深入讨论……
